# Math Methods
- Mathematical tasks on numbers.
- The Math object is static - no constructor.

| Methods      | Description              |
|--------------|--------------------------|
| Math.round() | 반올림                      |
| Math.ceil()  | 올림                       |
| Math.floor() | 버림                       |
| Math.trunc() | Returns the integer part |

## Examples

### Math.round()
```
Math.round(1.2);   // 1
Math.round(1.5);   // 2
Math.round(-1.8);  // -2
```

### Math.ceil()
```
Math.ceil(1.2);   // 2
Math.ceil(1.5);   // 2
Math.ceil(-1.8);  // -1
```

### Math.floor()
```
Math.floor(1.2);   // 1
Math.floor(1.5);   // 1
Math.floor(-1.8);  // -2
```

### Math.trunc()
```
Math.trunc(1.2);   // 1
Math.trunc(1.5);   // 1
Math.trunc(-1.8);  // -1
```
