# GitHub CLI
- GitHub CLI가 나왔다.
- pr, issue 등을 터미널에서 명령어로 확인 가능하다.
- 명령어 `gh`

## 설치
```
brew install gh
```

## 명령어 맛보기

#### clone
```git
gh repo clone {OWNER/REPO}
```

#### issue list
```git
gh issue list
```

#### pr 확인
```git
gh pr status
```

#### pr 생성
```git
gh pr create
```

##### * 자세한 사용법은 https://cli.github.com/manual/
