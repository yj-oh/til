# worktree 삭제
- worktree list 확인
```git
git worktree list
```
- worktree의 폴더 삭제
```git
rm -rf path/to/worktree
```
- worktree 삭제
```git
git worktree prune
```
- branch 삭제
```git
git branch -D <branch-name>
```
