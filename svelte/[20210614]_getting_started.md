# 시작하기
- https://svelte.dev/

## REPL
- https://svelte.dev/repl

## 프로젝트 생성
```
npx degit sveltejs/template {project_name}

cd {project_name}

npm install

npm run dev
```

## 💡 TypeScript 로 생성
```
npx degit sveltejs/template {project_name}

cd {project_name}

node scripts/setupTypeScript.js

npm install

npm run dev
```
- scripts 디렉토리가 삭제됨.
