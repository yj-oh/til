# \<dl>\<dt>\<dd>
```html
<dl>
    <dt>오늘</dt>
    <dd>지금 지나가고 있는 이날.</dd>
    <dt>내일</dt>
    <dd>오늘의 바로 다음 날.</dd>
</dl>
```
