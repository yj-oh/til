# vim 명령어

| 명령어      | 설명                 |
|----------|--------------------|
| dd       | 줄 삭제               |
| yy       | 줄 복사               |
| p        | 커서 뒤에 붙여넣기         |
| P        | 커서 앞에 붙여넣기         |
| u        | 되돌리기               |
| ctrl + r | 다시 실행              |
| a        | 커서 있는 줄 끝에서 입력 모드  |
| o        | 커서 있는 줄 아래에 빈 줄 삽입 |
| O        | 커서 있는 줄 위에 빈 줄 삽입  |
| 0        | 줄의 처음으로 이동 (공백 포함) |
| ^        | 줄의 처음으로 이동 (공백 제외) |
| $        | 줄의 끝으로 이동          |
| H        | 화면 맨 위로 이동         |
| L        | 화면 맨 아래로 이동        |
