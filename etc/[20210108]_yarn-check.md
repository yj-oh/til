# yarn-check
- package 정리하기
- https://github.com/yhnavein/yarn-check
>- 버전 오래 됐을 경우 알려줌
>- 사용되지 않는 종속성 알려줌 
>- ES6 `import from` 구문 지원

## 설치
```
npm install -g yarn-check
```

## 실행
```
yarn-check
```
