## 알고리즘
- 표현 및 저장된 데이터를 대상으로 하는 문제의 해결 방법

## 자료구조와 알고리즘
- 자료구조
```java
int[] arr = {1, 2, 3, 4, 5};
```
- 알고리즘
```java
for(int i = 0; i < arr.length; i++) {
    sum += arr[i];
}
```
- 자료구조에 따라 알고리즘은 달라진다.
- 알고리즘은 자료구조에 의존적이다.
